Exercise for reference: 

letters = ["a", "b", "c", "d", "e", "f", "g", "h", "i", "j"]

Answer: 

letters = ["a", "b", "c", "d", "e", "f", "g", "h", "i", "j"]
print(letters[:3])

Explanation: 

[:3]  is just a shortcut [0:3] . Both would do the job so you can mark this exercise as solved if you the output was correct.
